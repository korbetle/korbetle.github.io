Github Usage

Install Git

Ubnutu

{% codeblock lang:bash %} sudo apt-get install git-core {% endcodeblock %}

Windows

Download form and install.

Config

Global

git config --global user.name "Your Name"
git config --global user.email "Your Email"

Config color

git config --global color.ui "auto"

Alias

{% codeblock lang:bash %} git config –global alias.co checkout git config –global alias.br branch git config –global alias.ci commit git config –global alias.st status {% endcodeblock %}

Git ignore

Example

{% codeblock lang:bash %} DS_Store {% endcodeblock %}

Init a repo

{% codeblock lang:bash %} mkdir REPO cd REPO git init {% endcodeblock %}

{% codeblock lang:bash %} git init REPO {% endcodeblock %}

Create a remote repo

{% codeblock lang:bash %} git init --bare test.git # Clone repogit clone test.git {% endcodeblock %}

Clone a remote repo

{% codeblock lang:bash %} git clone url {% endcodeblock %}

Connect a local repo to remote repo

Update

{% codeblock lang:bash %} git pull {% endcodeblock %}

Create a branch

{% codeblock lang:bash %} git push origin NewBranch {% endcodeblock %}

Delete remote branch

{% codeblock lang:bash %} git push origin :DeleteBranch {% endcodeblock %}

Delete local branch

{% codeblock lang:bash %} git branch -d DeleteBranch {% endcodeblock %}

Commit

{% codeblock lang:bash %} git commit -a -m “some message” {% endcodeblock %}

Push

{% codeblock lang:bash %} git push {% endcodeblock %}

Switch to new branch

{% codeblock lang:bash %} git checkout BRANCH {% endcodeblock %}

Rename a branch

{% codeblock lang:bash %} git br -m old_branch_name new_branch_name {% endcodeblock %}

View branch

Local

{% codeblock lang:bash %} git branch {% endcodeblock %}

Remote

{% codeblock lang:bash %} git branch -r {% endcodeblock %}

All

{% codeblock lang:bash %} git branch -a {% endcodeblock %}

See local branch information

{% codeblock lang:bash %} cat .git/config {% endcodeblock %}

Merge

{% codeblock lang:bash %} git merge Branch {% endcodeblock %}

View change log

View local repo change log

{% codeblock lang:bash %} git log {% endcodeblock %}

View all branch log

{% codeblock lang:bash %} git status {% endcodeblock %}

Tag

Crate tag

{% codeblock lang:bash %} git tag [tag_name] [version] {% endcodeblock %}

List current tag

{% codeblock lang:bash %} git tag -l {% endcodeblock %}

Delete tag

{% codeblock lang:bash %} git tag -d [tag name] {% endcodeblock %}

Patch

{% codeblock lang:bash %}

git diff filename1 filename2 git diff > xxx.patch git apply xxx.patch

{% endcodeblock %}

Revert

{% codeblock lang:bash %} git revert git reset -hard {% endcodeblock %}
